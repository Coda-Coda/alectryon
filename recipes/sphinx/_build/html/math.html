MathJax in Sphinx¶

Using any math on a page causes Sphinx to automatically load MathJax: \(e^{i\pi} = -1\).

If you want to highlight pieces of code with MathJax, too, then see README.rst and the implementation of _static/mathjax_config.js:

Notation "\mathbb{N}" := nat.

Inductive nat : Set :=
    O : \mathbb{N} | S : \mathbb{N} -> \mathbb{N}

Arguments S _%nat_scope
